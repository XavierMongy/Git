Cheat Sheet Git

Créer

  • Créer un nouveau dépôt en local
    git init

  • Clôner un dépôt éxistant
    git clone ssh

Supprimer

  • Supprimer tous les changements dans le dossier de travail
    git reset --hard HEAD

  • Supprimer les changements dans un fichier spécifique
    git checkout HEAD "nomdufichier"

  • Faire revenir un commit
    git revert "commit"

Mise à jour et publication

  • Montrer les informations à propos du remote
    git remote show "remote"

  • Lister tous les remotes configurés
    git remote -v

  • Ajouter un nouveau remote
    git remote add "shortname" "url"

  • Télécharger les changements
    git pull "remote" "branch"

  • Publier les chamgements en local sur un remote
    git push "remote" "branch"

Historique des commit

  • Montrer tous les commit
    git log

  • Montrer les changements pour un fichier spécifique
    git log -p "fichier"

  • Qui a changé le fichier
    git blame "fichier"

Afficher les modifications

  • Afficher les fichiers modifiés dans l'espace de travail
    git status

  • Rendre les fichiers suivis
    git diff

  • Ajouter des changements au prochain commit
    git add

  • Faire un commit des changements
    git commit "message"
